# TextTalker

## Features
- Inro page (Host a meeting or join a meeting page)
- Name Feature(You can see names of participants)
- Video and Voice chat
- Live chat
- Invite Participants
- close/Open Video
- Mute/UnMute Audio



## Test

## How to run
- cd texttalker
- npm server.js
- Go to LocalHost:3030
